# 🔐 Random Password Generator

A simple, responsive, and professional Random Password Generator built with HTML, CSS, and JavaScript. This tool helps users generate secure, random passwords with customizable length, numbers, and symbols to enhance online security.

## 💻 Demo

The project can be run locally in any modern web browser.
OR
Live Demo # https://sannansarwer.github.io/js-password-generate/

## 📝 Features

- Generate strong, random passwords with customizable length.

- Option to include numbers and symbols.

- Copy generated password to clipboard with a single click.

- Reset all inputs to default.

- Fully responsive layout for mobile and desktop.

- Styled with Bootstrap 5 and custom CSS for a modern look.

## 🛠️ Technologies Used

- HTML5 – Structure of the page.

- CSS3 – Styling, including responsiveness and custom design.

- JavaScript (ES6) – Logic for password generation, copy functionality, and interactivity.

- Bootstrap 5 – Responsive grid system and UI components.

- Font Awesome – Icons for copy and success indicators.

## 🖱️ Usage

- Adjust the slider to select the desired password length.

- Check the Numbers or Symbols options if needed.

- Click Generate Random Password to create a secure password.

- Click the copy icon to copy the password to the clipboard.

- Use Reset to clear all inputs and selections.

## ⚡ Customization

- Change character sets: Modify letters, numbers, or symbols in script.js.

- Adjust slider range: Modify the min and max attributes in index.html for password length.

- Styling: Update style.css or Bootstrap classes for a different look.

## 📌 Notes

- If no checkboxes are selected, passwords will include only letters.

- Password length of 0 will result in an empty field.
